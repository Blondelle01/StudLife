 StudLife - Application de collecte et d’analyse de données

##  Présentation
StudLife est une application web développée dans le cadre d’un TP académique (INF232).  
Elle permet de collecter, stocker et analyser des données sur les étudiants via une interface simple et interactive.

---

##  Objectifs du projet
- Collecter des données utilisateur via un formulaire web
- Stocker les données dans le navigateur (localStorage)
- Afficher les données sous forme de tableau
- Calculer automatiquement des statistiques descriptives
- Visualiser les données à l’aide de graphiques

---

##  Fonctionnalités

###  1. Formulaire de collecte
L’utilisateur peut saisir les informations suivantes :
- Âge
- Sexe
- Filière
- Nombre d’heures d’étude par jour
- Utilisation d’internet (Oui / Non)

---

###  2. Affichage des données
Toutes les données enregistrées sont affichées dans un tableau structuré :
- Lecture claire
- Organisation en colonnes
- Mise à jour automatique

---

###  3. Statistiques automatiques
L’application calcule et affiche :
- Âge moyen des étudiants
- Nombre moyen d’heures d’étude
- Répartition hommes / femmes
- Nombre d’étudiants utilisant internet

---

###  4. Visualisation graphique
Grâce à **Chart.js**, les données sont représentées sous forme de graphiques :
- Diagramme en barres
- Répartition des catégories

---

###  5. Gestion des données
- Ajout de données via formulaire
- Stockage local (localStorage)
- Suppression complète des données (reset)

---

##  Structure du projet


index.html → Formulaire de saisie et enregistrement
data.html → Tableau des données enregistrées
stats.html → Statistiques + graphiques
